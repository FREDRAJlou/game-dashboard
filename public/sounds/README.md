# Victory Sound

Place a victory/champion sound effect as `victory.mp3` in this directory.

## Recommended Sources for Free Victory Sounds:

1. **Freesound.org** - https://freesound.org/search/?q=victory
2. **Pixabay** - https://pixabay.com/sound-effects/search/victory/
3. **Mixkit** - https://mixkit.co/free-sound-effects/win/

## Suggested Search Terms:
- "victory fanfare"
- "champion sound"
- "win celebration"
- "trophy unlock"
- "achievement sound"

## File Requirements:
- Format: MP3
- Duration: 1-3 seconds recommended
- File name: `victory.mp3`

The sound will play when users hover over the #1 ranked player in any group leaderboard.
